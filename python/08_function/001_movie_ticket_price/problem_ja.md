
# 001　映画館のチケット料金

## 映画館のチケット料金
- 次の条件で映画館のチケット料金を計算するプログラムを作成する
- 年齢が 0歳以上12歳以下 の場合、料金は 500円 とする
- 年齢が 13歳以上60歳以下 の場合、今日は月曜日なら 1800円、月曜日でなければ 2000円 とする
- 年齢が 61歳以上100歳以下 の場合、料金は 1500円 とする
- 上記のどれにも当てはまらない年齢が入力された場合は、不正な入力とする

### プログラムの仕様
- ticketPrice(age, isMonday) 関数を作成する
- age は年齢（整数）
- isMonday は月曜日なら True、それ以外は False
- 正しい年齢なら料金を返し、不正な場合は -1 を返す


## 実行結果
```
# 年齢50歳、月曜日のとき
年齢を入力してください: 50
今日は月曜日ですか？(y:はい n:いいえ): y
1800円です

# 年齢50歳、月曜日でないとき
年齢を入力してください: 50
今日は月曜日ですか？(y:はい n:いいえ): n
2000円です

# 年齢5歳、月曜日のとき
年齢を入力してください: 5
今日は月曜日ですか？(y:はい n:いいえ): y
500円です
```
