
# 010 電車の混雑状況

## 電車の混雑状況
- 6時から23時まで1時間ごとの電車の混雑状況を表示する

## 条件
- 7時から9時は「通勤ラッシュ」
- 17時から19時は「帰宅ラッシュ」
- それ以外は「通常運行」
- さらに、8時と18時は特に混雑するため、「★超混雑★」と付け加えること


## 実行結果
```
6時: 通常運行
7時: 通勤ラッシュ
8時: 通勤ラッシュ★超混雑★
9時: 通勤ラッシュ
10時: 通常運行
11時: 通常運行
12時: 通常運行
13時: 通常運行
14時: 通常運行
15時: 通常運行
16時: 通常運行
17時: 帰宅ラッシュ
18時: 帰宅ラッシュ★超混雑★
19時: 帰宅ラッシュ
20時: 通常運行
21時: 通常運行
22時: 通常運行
23時: 通常運行
```


## ヒント
for文、if/elif/else文を使い、「〇時: □□□□」の形式で表示する
