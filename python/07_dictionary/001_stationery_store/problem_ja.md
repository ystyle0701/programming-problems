
# 001　文房具店の在庫管理

## 文房具店の在庫管理
- 次の在庫データ（商品名、在庫数）を辞書で作る

| 商品名 | 在庫数 |
|----------|----------|
| ボールペン | 100 |
| ノート | 45 |
| 消しゴム | 80 |
| ホッチキス | 25 |

- 「ノートを何冊購入しますか？」と標準入力で受け取り、ノートの在庫数を減らす
- なお、在庫がない場合、「在庫がありません」とメッセージを表示し、在庫数は減らさない
- すべての商品情報を表示する

## 実行結果
```
# 10と入力
ノートを何冊購入しますか？10
---在庫状況---
ボールペン: 100個
ノート: 35個
消しゴム: 80個
ホッチキス: 25個

# 50と入力
ノートを何冊購入しますか？50
在庫がありません
---在庫状況---
ボールペン: 100個
ノート: 45個
消しゴム: 80個
ホッチキス: 25個
```
