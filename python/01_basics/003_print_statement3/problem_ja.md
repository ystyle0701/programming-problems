# 003 print()関数3　print関数の色々な書き方

## 問題文
1. print関数のendを使い、次のように表示する。
```
A-B-C-D
```

```python
# NGな書き方
print('A-B-C-D')
```
<br />

2. '*'を使い、次のように表示する。
```
*
**
***
****
*****
```
```python
# NGな書き方
print('*')
print('**')
print('***')
print('****')
print('*****')
```

<br />

3. 3つの数字の合計と平均を表示するプログラムを作る。
下のプログラムを完成させ、同じ出力結果となるように作ること。
```python
# プログラム
print('3 + 5 + 8 = ', )
print()
print('平均 = ', )
print()

# 出力結果
3 + 5 + 8 = 16
平均 = 5.333333333333333
```
