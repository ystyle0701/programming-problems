# 001 print()関数

## 問題文
1. あなたの名前を表示する（姓と名の間はスペース）  

2. あなたの名前を表示する（姓と名の間は改行）

3. 次のようにプロフィールを表示する
```
名前:佐藤
年齢:25
趣味:音楽を聴くこと
```

4. 次のように「Python!」という文字を連続して表示する
（+を使い文字列を連結させる）
```
Python!Python!Python!
```

5. 次の文を使い、>_< という顔文字を表示する
```
print('>')
print('_')
print('<')
```

## ヒント
シンプルなprint文の書き方の練習です。
