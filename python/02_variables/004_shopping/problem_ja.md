# 004　買い物

## 買い物
- たろうさんは1000円を持ってお菓子を買いに行きました
- 1個100円のチョコレートを買った後の所持金を表示し、さらに残ったお金でチョコレートがあと何個買えるか表示する

```
たろう さんの所持金 1000 円
たろう さんは チョコレート を買いました(金額 100 円)
たろう さんの所持金 900 円
残金で チョコレート を 9 個買えます
```
### 条件
- 以下の4つの変数を使用すること
- name (名前)
- money (所持金)
- price (商品の価格)
- item (商品名)

<br />

## ヒント
- 割り算（小数点以下切り捨て）を行いたいときは、//記号を使います
