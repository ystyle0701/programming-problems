
# 006　英語の成績

## 英語の成績
- ユーザーに英語のテストの点数（0〜100の整数）を入力させ、以下の基準に従って成績（A〜F）を判定し、点数と成績を表示する
- 出力には f文字列 を使用すること

| 点数の範囲 | 成績 |
|----------|----------|
| 90点以上 | A |
| 80点以上90点未満 | B |
| 70点以上80点未満 | C |
| 60点以上70点未満 | D |
| 60点未満 | F |


## 実行結果
```python
# 95と入力
英語の点数を入力してください: 95
点数: 95点, 成績: A

# 80と入力
英語の点数を入力してください: 80
点数: 80点, 成績: B

# 59と入力
英語の点数を入力してください: 59
点数: 59点, 成績: F
```
