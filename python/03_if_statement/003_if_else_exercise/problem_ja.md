## 問題
1. 年齢を入力し、18歳以上であれば「投票できます」、それ以外は「投票できません」と表示する
```python
# 17と入力

```
<br />

2. 英語の点数（0〜100）を入力し、80点以上なら「合格です」、それ以外は「不合格です」と表示する
```python
# 79と入力
英語の点数を入力してください：79
不合格です
```

<br />

3. 年齢を入力し、13歳以下であれば「割引対象です」、それ以外は「割引対象外です」と表示する

```python
# 50と入力

```
