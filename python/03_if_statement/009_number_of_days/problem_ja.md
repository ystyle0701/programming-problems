
# 009　月の日数

## 月の日数
- ユーザーに1〜12の整数を入力させ、対応する月の日数を表示する
- ただし、うるう年は考慮せず、2月は28日までとする

| 月 | 表示内容 |
|----------|----------|
| 1,3,5,7,8,10,12 | 31日まであります |
| 4,6,9,11 | 30日まであります |
| 2 | 28日まであります |
| 上記以外 | 分かりません |


## 実行結果
```
# 1と入力
調べたい月を入力してください(1～12)：1
31日まであります

# 2と入力
調べたい月を入力してください(1～12)：2
28日まであります

# 20と入力
調べたい月を入力してください(1～12)：20
分かりません
```
