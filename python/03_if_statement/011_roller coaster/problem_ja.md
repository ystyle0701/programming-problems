
# 011　ジェットコースター利用前の確認

## ジェットコースター利用前の確認
- ジェットコースターに乗る前の条件確認を行うプログラムを作成する
- 以下の利用制限に従い、ユーザーに質問し、乗車の可否を判定する
- 車椅子を利用している方は、利用できません
- 身長120cm未満または190cm以上の方は、利用できません
