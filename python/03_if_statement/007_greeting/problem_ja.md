
# 007　挨拶

## 挨拶
- ユーザーに現在の時間（0～23の整数）を入力させ、以下のルールに従い、時間帯に応じて適切な挨拶を表示する
- 出力には f文字列 を使用すること

| 時間の範囲 | 挨拶 |
|----------|----------|
| 6時～11時 | おはよう |
| 12時～17時 | こんにちは |
| 18時～21時 | こんばんは |
| 上記以外 | お疲れ様 |


## 実行結果
```python
# 6と入力
時間を入力してください:6
6時: おはよう

# 15と入力
時間を入力してください:15
15時: こんにちは

# 23と入力
時間を入力してください:23
23時: お疲れ様
```
