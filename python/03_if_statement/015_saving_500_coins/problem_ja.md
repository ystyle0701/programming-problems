
# 015　500円玉のみで貯金

## 500円玉のみで貯金
- ユーザーに目標金額を入力させ、必要な500円硬貨の数を求める

### 条件
- 金額が負の場合「正の金額を入力してください」と表示する
- 金額が0の場合「0円なので硬貨は必要ありません」と表示する
- 金額が500で割り切れる場合「500円硬貨は〇枚必要です」と表示する
- 上記以外、「500円硬貨以外が必要です」と表示する


## 実行結果
```
# 1500と入力
目標金額を入力してください: 1500
500円硬貨は3枚必要です

# 1600と入力
目標金額を入力してください: 1600
500円硬貨以外が必要です

# -100と入力
目標金額を入力してください: -100
正の金額を入力してください

# 0と入力
目標金額を入力してください: 0
0円なので硬貨は必要ありません
```
