
# 014　アプリの会員割引

## アプリの会員割引
- アプリ会員には割引があり、年齢により割引率が変わるシステムがある
- 以下の条件に従い、割引率を表示させること
- アプリ会員でない：割引なし
- アプリ会員である：60歳以上は5％割引、それ以外は3％割引


## 実行結果
```
# アプリ会員はy、年齢は60と入力
アプリ会員ですか？(y:はい/n:いいえ)y
年齢を入力してください: 60
5%割引です

# アプリ会員はy、年齢は10と入力
アプリ会員ですか？(y:はい/n:いいえ)y
年齢を入力してください: 10
3%割引です

# アプリ会員はnと入力
アプリ会員ですか？(y:はい/n:いいえ)n
割引はありません

# アプリ会員は10と入力
アプリ会員ですか？(y:はい/n:いいえ)10
入力に誤りがあります
```
