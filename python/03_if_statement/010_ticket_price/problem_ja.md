
# 010　チケットの金額

## チケットの金額
- ユーザーに年齢を入力させ、以下の条件で料金を表示する
- その後、購入したいチケットの枚数を入力させ、合計金額を計算・表示する
- 出力には f文字列 を使用すること

| 条件 | 1枚あたりの料金 |
|----------|----------|
| 18歳以上（大人） | 1000円 |
| 18歳未満（子ども） | 500円 |


## 実行結果
```
# 年齢は20、枚数は5と入力
年齢を入力してください：20
大人料金です(1枚 1000円)
チケットの枚数を入力してください：5
料金は5000円です

# 年齢は10、枚数は3と入力
年齢を入力してください：10
子ども料金です(1枚 500円)
チケットの枚数を入力してください：3
料金は1500円です
```
