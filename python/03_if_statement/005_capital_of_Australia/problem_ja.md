
# 005　オーストラリアの首都

## オーストラリアの首都
- ユーザーに「オーストラリアの首都はシドニーです」という問いを出し、ユーザーが y（正しい）または n（間違い）で答えると、それに応じて以下のように応答するプログラムを作成する
- y と答えた場合：「違います、正解はキャンベラです」と表示する
- n と答えた場合：「正解です」と表示する
- それ以外の入力があった場合：「入力に誤りがあります」と表示する

## 実行結果
```python
# yと入力
オーストラリアの首都はシドニーです(y:正しい n:間違い)y
違います、正解はキャンベラです

# nと入力
オーストラリアの首都はシドニーです(y:正しい n:間違い)n
正解です
```
