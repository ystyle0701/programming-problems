
# 012　クレジットカード番号のマスキング

## クレジットカード番号のマスキング
- ユーザーにクレジットカード番号を入力させ、下4桁以外は*でマスキングする
- 入力された数値が16桁以外の場合、「16桁で入力してください」と表示する
- 出力には f文字列 を使用すること


## 実行結果
```
# 1234567890123456と入力
クレジットカード番号を入力してください: 1234567890123456
************3456

# 55555と入力
クレジットカード番号を入力してください: 55555
16桁で入力してください
```

### ヒント
下4桁の文字列を取得したい場合、文字列[-4:]と書きます
