
# 013　評価判定

## チケットの金額
- ユーザーに試験の点数と出席日数を入力させ、以下の条件で判定を表示する
- 試験の点数は0～100
- 出席日数は0～20
- 出力には f文字列 を使用すること

| 条件 | 判定結果 |
|----------|----------|
| 点数が80点以上かつ出席18日以上 | 優秀 |
| 点数が60点以上かつ出席15日以上 | 合格 |
| 点数が60点以上かつ出席15日未満 | 出席不足で不合格 |
| 点数が60点未満かつ出席15日以上 | 点数不足で不合格 |
| 点数が60点未満かつ出席15日未満 | 両方不足で不合格 |


## 実行結果
```
# 点数は80、出席日数は18と入力
点数を入力してください(0-100): 80
出席日数を入力してください(0-20): 18
点数: 80点, 出席日数: 18日
判定: 優秀

# 点数は50、出席日数は20と入力
点数を入力してください(0-100): 50
出席日数を入力してください(0-20): 20
点数: 50点, 出席日数: 20日
判定: 点数不足で不合格
```
